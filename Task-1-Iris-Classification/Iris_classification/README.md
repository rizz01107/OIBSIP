## üìù Project Overview
This repository contains the solution for **Task 1** of the Oasis Infobyte Internship. The goal of this project is to classify Iris flowers into three species: **Setosa, Versicolor, and Virginica** based on their physical measurements (sepal length, sepal width, petal length, and petal width).

## üöÄ Key Features
* **Exploratory Data Analysis:** Comprehensive data visualization using Seaborn.
* **Machine Learning Model:** Implemented a **Decision Tree Classifier**.
* **Performance:** Achieved **100% Accuracy** on the test dataset.
* **Interactive Web App:** Developed a user-friendly UI using **Streamlit** for real-time predictions.
